# CineBook - MERN Movie Ticket Booking

CineBook is a production-ready MERN stack movie ticket booking platform with JWT auth, seat selection, and an admin panel. It uses React + Vite + Tailwind on the frontend and Node.js + Express + MongoDB on the backend.

## Features

- JWT-based authentication with roles (user/admin)
- Search and filter movies by title, genre, and language
- Showtime selection with 8x10 interactive seat layout
- Seat type pricing: Regular ₹150, Premium ₹200, VIP ₹300
- Booking conflict prevention at API level
- User booking history and cancellation
- Admin movie management and booking overview
- Responsive glassmorphism UI with purple gradient theme

## Tech Stack

- Frontend: React, Vite, Tailwind CSS, React Router v6, React Toastify, React Icons
- Backend: Node.js, Express, MongoDB (Atlas), Mongoose, JWT, bcrypt
- Tooling: ES6 modules, dotenv, nodemon
